Publish Docker image to GitHub Container Registry (ghcr.io) via CI, and document how to run from pre-built image in docker-compose.yml.
</objective>

<execution_context>
### Plan 06-01: GHCR publishing

**Task 6.1: Update CI workflow**

Add a `push` job after `build`:

```yaml
  push:
    runs-on: ubuntu-latest
    needs: [test, build]
    if: github.event_name == 'push' && github.ref == 'refs/heads/main'
    permissions:
      contents: read
      packages: write
    steps:
      - uses: actions/checkout@v4
      - name: Log into ghcr.io
        uses: docker/login-action@v3
        with:
          registry: ghcr.io
          username: ${{ github.actor }}
          password: ${{ secrets.GITHUB_TOKEN }}
      - name: Build and push Docker image
        run: |
          REPO=ghcr.io/${{ github.repository_owner }}/bg-price-monitor
          REPO_LOWER=$(echo "$REPO" | tr '[:upper:]' '[:lower:]')
          docker compose build
          docker tag bg-price-monitor:latest ${REPO_LOWER}:latest
          docker tag bg-price-monitor:latest ${REPO_LOWER}:sha-${GITHUB_SHA::7}
          docker push ${REPO_LOWER}:latest
          docker push ${REPO_LOWER}:sha-${GITHUB_SHA::7}
```

**Task 6.2: Update docker-compose.yml**

Add commented GHCR image example above the `build:` block:

```yaml
    # Build from source (default):
    build:
      context: .
      dockerfile: docker/price-monitor/Dockerfile
    # Or use pre-built image from GitHub Container Registry:
    # image: ghcr.io/<your-username>/bg-price-monitor:latest
    # For the above, comment out the `build:` block above and uncomment `image:`.
    image: bg-price-monitor:latest
```
