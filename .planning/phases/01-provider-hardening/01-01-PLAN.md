<objective>
Split the monolithic `monitor.py` parsers into focused `price_monitor/parsers/` modules with real HTML fixture tests and per-provider rate limits plus retry budgets. Backward-compatible wrappers in `monitor.py` ensure zero breakage.
</objective>

<execution_context>
### Plan 01-01: Create parser package and fixtures

**Task 1.1: Create `price_monitor/parsers/_fetch.py`**
- Shared HTTP helper: `fetch_with_retry(url, provider_name, rate_limit_seconds=10, max_retries=3)`
- Rate limiting via `_last_request: dict[str, float]` ‚Äî sleeps if needed
- Retry on 5xx, 429, connection errors with exponential backoff 1s‚Üí2s‚Üí4s
- Returns `requests.Response` or raises after all retries exhausted

**Task 1.2: Create parser modules**

Move existing logic from `monitor.py` into focused modules:

`price_monitor/parsers/bgoperator.py`:
- `parse_offers(html: str) -> list[Offer]`
- `extract_hotel_name(html: str, url: str) -> str | None`

`price_monitor/parsers/leveltravel.py`:
- `parse_external_price(html: str, url: str) -> ExternalPrice`
- `extract_hotel_name(html: str, provider: str) -> str | None`

`price_monitor/parsers/travelata.py`:
- `parse_external_price(html: str, url: str) -> ExternalPrice`
- `extract_hotel_name(html: str, provider: str) -> str | None`

Move shared helpers to `parsers/__init__.py` or `_fetch.py`:
- `parse_int()`, `decode_js_string()`, `first_query_value()`, `provider_from_url()`

**Task 1.3: Save real HTML fixtures**

Save current actual HTML responses as `tests/fixtures/bgoperator.html`, `leveltravel.html`, `travelata.html`. These replace inline HTML strings in tests.

**Task 1.4: Backward-compatible wrappers in `monitor.py`**

Update `monitor.py` functions to delegate to new modules:
```python
from price_monitor.parsers import bgoperator, leveltravel, travelata

def parse_offers(html): return bgoperator.parse_offers(html)
def parse_external_price(html, url):
    provider = provider_from_url(url)
    if "level.travel" in provider.lower():
        return leveltravel.parse_external_price(html, url)
    return travelata.parse_external_price(html, url)
```
